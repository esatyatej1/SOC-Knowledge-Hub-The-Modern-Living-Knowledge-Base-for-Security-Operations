# SOC Knowledge Hub

A modern, structured, and curated SOC knowledge platform for security operations teams. This is a living knowledge base for detection engineering, incident response, threat intelligence, and SOC leadership.

## Why this project exists
Most “awesome” repositories are useful but flat. They provide links, but not enough structure, operational context, and role-based guidance.

SOC Knowledge Hub creates an operator-grade, curated collection of the best resources, templates, and best practices in the field.

## Core Sections
- **[Fundamentals](./docs/fundamentals/soc-basics.md)**: SOC definitions, roles, and core functions.
- **[SOC Architecture](./docs/architecture/soc-architecture.md)**: Design patterns, telemetry pipelines, and technology stacks.
- **[Detection Engineering](./docs/detection-engineering/detection-lifecycle.md)**: Sigma, KQL, detection lifecycle, and monitoring.
- **[Threat Intelligence](./docs/threat-intelligence/ti-basics.md)**: OSINT, TI formats, and lifecycle management.
- **[Incident Response](./docs/incident-response/ir-lifecycle.md)**: NIST/SANS frameworks, containment, and forensics.
- **[SOAR and Automation](./docs/soar/soar-basics.md)**: Orcherstration, playbooks, and automation strategy.
- **[SOC Tools](./docs/tools/siem.md)**: SIEM, XDR, EDR, NDR, and other platform references.
- **[Management and KPIs](./docs/management/kpis.md)**: Metrics, staffing models, and governance.
- **[Learning Paths](./docs/learning-paths/l1-analyst.md)**: Tailored paths for L1, L2, L3 analysts.
- **[Live Website Monitoring](./docs/live-website-monitoring/uptime.md)**: External security exposure and uptime.
- **[Templates and Checklists](./docs/templates/incident-template.md)**: Practical documents you can use today.

## Who this is for
- L1 / L2 / L3 SOC Analysts
- Detection Engineers
- Threat Hunters & Incident Responders
- SOC Managers & Architects
- GRC and Compliance Teams

## How to navigate
Simply browse the `docs/` folder. Every page is a curated guide with directly accessible links and practical summaries. No scripts, no databases—just pure knowledge.
