# Awesome SOC Must-Reads

A curated list of the most authoritative and useful resources for SOC professionals.

## Frameworks & Standards
- **[MITRE ATT&CK](https://attack.mitre.org/)**: The globally-accessible knowledge base of adversary tactics and techniques.
- **[NIST Incident Handling Guide (SP 800-61)](https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.800-61r2.pdf)**: Foundation for enterprise IR plans.
- **[SOC-CMM](https://www.soc-cmm.com/)**: A model for measuring and improving SOC maturity.

## Knowledge Bases
- **[Awesome SOC (GitHub)](https://github.com/SOC-Community/Awesome-SOC)**: A massive collection of SOC resources (The inspiration for this hub).
- **[The Detection Engineering Handbook](https://github.com/Detection-Engineering-Handbook/handbook)**: In-depth guide for detection engineers.
- **[OWASP Web Security Testing Guide](https://owasp.org/www-project-web-security-testing-guide/)**: The industry standard for web security testing.

## Training & Labs
- **[TryHackMe SOC Level 1](https://tryhackme.com/path/outline/soclevel1)**: Hands-on learning for beginners.
- **[Antisyphon Training](https://www.antisyphonsecurity.com/)**: Practical, community-focused security training.
- **[Blue Team Labs Online](https://blueteamlabs.online/)**: Gamified platform for defensive security skills.

## Newsletters & Communities
- **[SANS NewsBites](https://www.sans.org/newsletters/newsbites/)**: High-level weekly security news.
- **[TDR (Threat Detection Report)](https://redcanary.com/threat-detection-report/)**: Red Canary's deep dive into real-world threats.
- **[SOC Community on Discord](https://discord.gg/soc-community)**: A place to discuss tradecraft with other analysts.
