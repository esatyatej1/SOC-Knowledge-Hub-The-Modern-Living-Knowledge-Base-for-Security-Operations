# SOC Knowledge Hub

Welcome to the **SOC Knowledge Hub**, a modern, structured, enterprise-grade SOC knowledge platform.

This platform is designed for security operations teams, detection engineers, incident responders, SOC managers, and cybersecurity architects.

## Get Started

- **[Fundamentals](./fundamentals/soc-basics.md)**: Learn the core concepts of SOC operations.
- **[Architecture](./architecture/soc-architecture.md)**: Explore modern SOC design patterns.
- **[Detection Engineering](./detection-engineering/detection-lifecycle.md)**: Master the art of detection.
- **[Incident Response](./incident-response/ir-lifecycle.md)**: Structured approach to handling security breaches.
- **[Live Website Monitoring](./live-website-monitoring/uptime.md)**: Secure your external digital presence.

## Features

- **Role-based Navigation**: Content tailored for L1, L2, L3, and Managers.
- **Structured Data**: Every resource is tracked in JSON for easy maintenance.
- **Automation**: CI/CD for link checking and freshness scoring.
- **Vendor Neutral**: Focus on concepts and open standards.
