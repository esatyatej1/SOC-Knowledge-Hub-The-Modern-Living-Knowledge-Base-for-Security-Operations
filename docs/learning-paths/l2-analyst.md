# L2 Analyst Learning Path (Advanced Responder)

## Role Description
The L2 Analyst handles complex investigations escalated from L1. They perform deep forensics, build initial timelines, and execute advanced containment actions.

## Advanced Skills
- **Forensics (Disk & Memory):** Analyzing MFT, Shimcache, and Volatility dumps.
- **Malware Analysis:** Basic static/dynamic analysis (Cuckoo, Joe Sandbox).
- **Log Correlation:** Joining telemetry from EDR, NDR, and Cloud logs (e.g., AzureAD vs. Office365).
- **Advanced Querying:** Mastering KQL/SPL sub-queries and joins.

## Core Training Resources
1. **[SANS SEC504: Hacker Tools & Incident Response](https://www.sans.org/cyber-security-courses/hacker-tools-techniques-exploits-incident-handling/)**: Highly recommended industry standard.
2. **[BTL2 (Blue Team Level 2)](https://securityblue.team/)**: Practical certification for intermediate analysts.
3. **[Malware Analysis for Everyone](https://github.com/target/malware-analysis-for-everyone)**: A free guided resource.

## Quarterly Goals
### Q1: Forensic Mastery
- Complete 5 memory analysis labs using Volatility.
- Learn how to extract IoCs from a disk image.

### Q2: Malware Triage
- Set up a malware lab environment.
- Perform a dynamic analysis on a ransomware sample.

### Q3: Complex Reporting
- Write a full Incident Report for a simulated ransomware attack.
- Brief the SOC Manager on a technical finding.
