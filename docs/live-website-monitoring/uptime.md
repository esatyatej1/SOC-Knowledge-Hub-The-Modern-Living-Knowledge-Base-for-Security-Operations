# Live Website Monitoring

## Why Monitor External Websites?
A company's website is its digital storefront. Security operations must ensure it is not only online but also secure from attacks, defacement, and impersonation.

## Core Monitoring Areas
1. **Uptime & Performance:** Monitoring from multiple geographic locations.
2. **SSL/TLS Health:** Checking for expiring certificates and weak encryption protocols.
3. **Security Headers:** Validating CSP, HSTS, X-Frame-Options, etc., to prevent web attacks.
4. **DNS Security:** Monitoring for unauthorized DNS changes (hijacking).
5. **Exterior Exposure / ASM:** Identifying subdomains and services that shouldn't be public.
6. **Domain Impersonation:** Looking for look-alike domains used in phishing.

## Key Tools & References
- **[UptimeRobot](https://uptimerobot.com/):** Simple uptime monitoring.
- **[SSLLabs](https://www.ssllabs.com/ssltest/):** Deep analysis of SSL configuration.
- **[SecurityHeaders.com](https://securityheaders.com/):** Quick scan for missing security headers.
- **[OWASP ZAP](https://www.zaproxy.org/):** Open-source web vulnerability scanner.
