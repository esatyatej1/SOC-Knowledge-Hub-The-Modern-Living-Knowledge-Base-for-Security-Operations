# SOC Metrics and KPIs

## Why Metrics Matter
"You cannot manage what you cannot measure." Metrics help SOC managers justify budgets, identify bottlenecks, and demonstrate value to leadership.

## Key Performance Indicators (KPIs)
- **MTTD (Mean Time to Detect):** Average time from the start of an incident to detection.
- **MTTR (Mean Time to Respond/Resolve):** Average time from detection to mitigation.
- **SLA Compliance:** Percentage of incidents handled within predefined timeframes.
- **False Positive Rate:** Percentage of alerts that are not true security incidents.
- **Analyst Utilization:** Balance of workload across the team to prevent burnout.
- **Case Backlog:** Number of incidents currently waiting for investigation.

## Operational Metrics
- **Log Source Health:** Percentage of expected logs arriving in the SIEM.
- **Detection Coverage:** Mapping detections to the MITRE ATT&CK matrix.
- **Threat Intel Hit Rate:** How many IoCs from TI feeds actually triggered alerts.
