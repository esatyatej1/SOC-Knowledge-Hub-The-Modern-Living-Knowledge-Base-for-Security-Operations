# SOC Tools - SIEM / XDR

## SIEM (Security Information and Event Management)
The SIEM is the historical center of the SOC. It provides real-time analysis of security alerts generated by applications and network hardware.

## XDR (Extended Detection and Response)
XDR is the evolution of SIEM and EDR. It integrates telemetry from endpoints, network, and cloud into a single unified platform for detection and response.

## Key Selection Criteria
- **Ingestion Costs:** How much does it cost to send data to the platform?
- **Query Language Complexity:** How easy is it to write detections?
- **Integration Ecosystem:** Does it support your current stack?
- **Community Support:** Are there pre-built rules and playbooks available?

## Market Leaders
- **Microsoft Sentinel:** Cloud-native SIEM/SOAR.
- **Splunk:** The original "Big Data" SIEM.
- **Elastic Security:** High-performance, search-driven SIEM.
- **CrowdStrike Falcon:** XDR leader focusing on endpoint health.
